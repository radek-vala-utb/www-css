Media queries

Media query umožní měnit vzhled podle šířky obrazovky. Zde je ukázka přístupu mobile first.

Klíčové vlastnosti

Zápis: @media (...). Časté podmínky: min-width, max-width, orientation.

Mobile first obvykle začíná základním stylem a širší obrazovky doplňuje přes @media (min-width: ...).
