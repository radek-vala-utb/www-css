CSS proměnné

Proměnné zjednodušují opakované používání barev, velikostí i rozestupů.

Klíčové vlastnosti

Definice: vlastní vlastnosti jako --primary-color nebo --space-md.

Použití: funkce var(--nazev-promenne), případně záložní hodnota var(--x, red).

Dokumentace: MDN: Custom properties

Panel používá hodnoty z :root

Když změníš proměnnou --primary-color, změní se více míst najednou.
