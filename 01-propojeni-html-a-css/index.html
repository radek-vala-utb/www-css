Propojení HTML a CSS

Tento příklad ukazuje tři způsoby, jak propojit HTML a CSS: inline, internal a external.

Klíčové vlastnosti

HTML prvky: style, <style>, <link rel="stylesheet">.

Možnosti použití: inline styl pro jeden prvek, internal CSS pro jednu stránku, external CSS pro více stránek.

Dokumentace: MDN: What is CSS?
