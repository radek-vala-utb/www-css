Box model a jednotky

Každý HTML prvek lze chápat jako obdélník. Obsahuje obsah, vnitřní odsazení, rámeček a vnější odsazení.

Klíčové vlastnosti

Box model: width, padding, border, margin, box-sizing.

Jednotky: px, %, rem, em, vw, vh.

Dokumentace: MDN: Box model

Oranžová plocha je obsah, světlý prostor uvnitř je padding, tmavy ramecek je border a volne místo okolo je margin.
